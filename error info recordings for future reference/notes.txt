How to update a row in a table using entityframework core
will automatically update based on the id in the obj object which is passed from the frontend.


when performing an update make sure that the id in obj is populated. 

        [HttpPost]
        public IActionResult Edit(Category obj)
         {
           
            if (ModelState.IsValid)
            {
                _db.Categories.Update(obj); // inside obj object otherwise a new row will be added.
                _db.SaveChanges();
                return RedirectToAction("Index");
            }
            return View();
        }
